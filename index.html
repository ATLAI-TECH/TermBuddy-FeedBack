Up and Running in Four Steps

No complicated setup. No manual configuration. Just tap and go.

1

Add Your Server

Enter your server's IP address and SSH credentials. That's all TermBuddy needs to get started.

2

Automatic Installation

TermBuddy installs its lightweight Go server on your machine automatically. No SSH-ing in to set things up.

3

Create Your Buddies

Each Buddy is a persistent terminal session. Create one for monitoring, another for deployments, another for logs.

4

Chat Away

Send commands, get results, ask AI for help. All in a friendly chat interface. From the sofa, the cafe, or the train.

Security Without Compromise

Your data stays yours. Always.

🔐

No Intermediate Servers

Direct connection from your device to your server. Your data never passes through anyone else's infrastructure.

🛡️

SSH + TLS Encryption

SSH Tunnel mode encrypts through SSH. Direct mode uses TLS with certificate pinning. Double the protection.

🔑

Keychain Storage

All credentials stored in the system Keychain. Token-based API authentication with automatic rate limiting.
